Lucky Money is a simple betting game where a player places a bet and tries to cash out before losing.
## Features
- Betting system with validation
- Quick bet selection using ChoiceChips
- Custom bet input
- Balance check and minimum bet validation
- Cashout mechanic
- Bottom sheet game settings
- Modular widget structure
